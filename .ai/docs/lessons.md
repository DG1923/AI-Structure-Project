# lessons.md

> Architectural, performance, and security lessons accumulated from this project.
> AI reads this at session start. Review before working on related areas.
> Session-level lessons live in `tasks/lessons.md` — graduate them here when they apply broadly.

---

## Format

```
## [YYYY-MM-DD] — [Short title]

**Context:** What feature/task was this?
**Mistake:** What went wrong?
**Root cause:** Why did it happen?
**Rule:** What rule prevents this in the future?
**Category:** performance | error-handling | architecture | security | algorithm
```

---

## Categories index

- `performance` — algorithm choices, N+1, indexes, caching, query optimization
- `error-handling` — deadlocks, timeouts, retry logic, silent failures
- `architecture` — structure decisions, coupling, boundaries, responsibility
- `security` — auth, input validation, data leakage, secret handling
- `algorithm` — data structure choices, complexity tradeoffs, batch vs sequential
